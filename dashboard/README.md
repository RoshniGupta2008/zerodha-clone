# Zerodha Clone

A full-stack stock trading platform clone built with the MERN stack.

## 🔗 Live Links
- Frontend: (apna frontend URL daalo)
- Dashboard: https://zerodha-dashboard-6pzb.onrender.com
- Backend: https://zerodha-backend-n4yb.onrender.com

## 🛠️ Tech Stack
- **Frontend:** React, Vite, React Router
- **Backend:** Node.js, Express.js
- **Database:** MongoDB Atlas
- **Charts:** Chart.js, Recharts

## ✨ Features
- Live Holdings with P&L calculation
- Buy/Sell orders with real-time backend save
- Portfolio Overview chart
- Watchlist with search filter
- Orders history

## 🚀 Run Locally

### Backend
```bash
cd backend
npm install
# create .env file with MONGO_URL and FRONTEND_URL
npm start
```

### Dashboard
```bash
cd dashboard
npm install
# create .env file with VITE_API_URL
npm run dev
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```
